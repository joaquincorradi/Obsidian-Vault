### Estructura básica
- Relación:
- Tupla: es una fila de una tabla relacional.
- Valor: componente de una tupla.
- Atributo: es un atributo de una tabla relacional.
- Dominio: conjunto de valores que puede tomar un atributo.

### Operaciones fundamentales

| Operador             | Qué hace                                                     |
| -------------------- | ------------------------------------------------------------ |
| Selección ($\sigma$) | Filtra filas (tuplas) que cumplen una condición (predicado). |
| Proyección ($\Pi$)   | Elige columnas                                               |
